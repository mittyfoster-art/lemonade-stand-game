### Technical Details

#### Animation Components (src/components/CountUpAnimation.tsx)
- `CountUpAnimation`: Animates numbers from start to target
  - Props: target, duration, prefix, suffix, decimals, className, start, useGrouping
  - Uses requestAnimationFrame for smooth 60fps animation
  - Intl.NumberFormat for localized number formatting
- `AnimatedProgressBar`: Animates progress bar fill
  - Props: value (0-100), duration, className, colorClass
  - Custom color classes supported
- `BudgetChangeAnimation`: Shows budget transitions with color flash
  - Props: from, to, duration, className
  - Three phases: showing-before, transitioning, showing-after
